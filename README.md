# 📝 Full Stack Todo List App

## 🚀 Overview

This is a full-stack Todo List application built with:

* Frontend: React
* Backend: Django (API)

## ✨ Features

* Add tasks
* Delete tasks
* View all tasks
* Backend API integration

## 📁 Project Structure

/frontend → React app
/backend → Django backend

## ⚙️ Setup Instructions

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
python manage.py runserver
```
